=== Blackbox.cool ===
Contributors: scott@blackbox.cool
Requires at least: 4.0
Tested up to: 4.5.3
Stable tag: 0.1

A convenience plugin for adding Blackbox widgets to a WordPress installation.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/blackbox-cool` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Copy/paste your widget's shortcode from the Blackbox dashboard, and paste it into your WordPress post or page.
